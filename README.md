# Cocky - simulace opticke lavice

Jednoducha webova simulace opticke lavice:

- vlevo je zdroj svetla, ktery vysila 5 laserovych paprsku,
- do drahy paprsku lze vkladat spojky i rozptylky,
- u kazde cocky lze menit polohu a ohniskovou vzdalenost (v centimetrech).

## Spusteni

Staci otevrit soubor `index.html` v prohlizeci.
